# Product Image Brief Generator Checklist

- [ ] Defines the goal of each image
- [ ] Keeps platform rules in mind
- [ ] Keeps main image compliant
- [ ] Uses short image text
- [ ] Avoids fake badges and certifications
- [ ] Avoids unsupported specs
- [ ] Includes shot list
- [ ] Includes props/background guidance
- [ ] Can be executed by a designer or photographer
