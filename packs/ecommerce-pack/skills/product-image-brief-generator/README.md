# Product Image Brief Generator

Creates product image briefs that designers and photographers can execute.

## Suitable Scenarios

- Main image planning
- Lifestyle image planning
- Feature image design
- Size/spec image creation
- Comparison image planning
- A+ content planning

## Steps

1. Provide product facts, platform, customer, features, benefits, and style.
2. Add image count and compliance restrictions.
3. Generate briefs.
4. Review platform image rules before production.

## Input Fields

Product, platform, target customer, features, benefits, specs, brand style, image count, competitor image notes, and restrictions.

## Output Fields

Main image, lifestyle image, feature image, size/spec image, comparison image, problem-solution image, A+ content image, text suggestions, shot list, props/background, and avoid list.

## Common Mistakes

- Adding text or badges to marketplace main images when prohibited
- Showing unsupported certifications
- Creating image text that is too long

## Compliance Reminder

Check marketplace image rules and verify every displayed claim before publishing.
