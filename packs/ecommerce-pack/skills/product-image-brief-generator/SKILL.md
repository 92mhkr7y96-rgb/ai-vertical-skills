# Skill: product-image-brief-generator

## 中文注释

这个 skill 用于生成商品图片设计或拍摄 Brief，覆盖主图、场景图、功能图、尺寸图、对比图、问题解决图和 A+ 内容图。Brief 应能交给设计师或摄影师执行，同时遵守平台图片规则，不添加虚假徽章、认证、评价数量、商标或未经证明的参数。

## Purpose

Create actionable design and photography briefs for ecommerce product images.

## Best for

Sellers, designers, photographers, and operators planning main images, feature images, lifestyle scenes, comparison images, and A+ content.

## Not for

Adding noncompliant main-image elements, fake certifications, fake awards, unsupported specs, or misleading before-after claims.

## Input

- Product
- Platform
- Target customer
- Key features
- Main benefits
- Materials/specs
- Brand style
- Image count
- Competitor image notes
- Compliance restrictions

## Output

1. Main image brief
2. Lifestyle image brief
3. Feature image brief
4. Size/spec image brief
5. Comparison image brief
6. Problem-solution image brief
7. A+ content image brief
8. Image text suggestions
9. Shot list
10. Props/background suggestions
11. Things to avoid

## Process

1. Define the goal of each image.
2. Match image type to platform rules.
3. Turn features and benefits into visual scenes.
4. Draft short image text suggestions.
5. List shots, props, backgrounds, and avoid items.

## Quality Bar

- Supports ecommerce conversion.
- Defines each image's purpose.
- Keeps main images compliant.
- Uses short and clear image text.
- Avoids unsupported certifications and specs.
- Can be handed to a designer or photographer.

## Example Input

Product: rechargeable camping lantern
Platform: Amazon and Shopify
Key features: hanging hook, adjustable brightness, compact body
Restrictions: no waterproof certification claim

## Example Output

Main image brief: product only on clean white background for Amazon; show front angle and hanging hook clearly; avoid props, badges, and unsupported claims.

## Safety / Compliance Notes

Respect platform image rules. Do not add fake badges, certifications, review counts, trademarks, or unsupported performance claims.
