# Product Positioning Analyzer

Helps sellers define target customers, pain points, motivations, differentiation, and messaging angles.

## Suitable Scenarios

- New product planning
- Brand positioning
- Listing strategy
- Ad angle planning
- Product image strategy

## Steps

1. Provide product, category, customer, competitors, and differentiators.
2. Add sales channel and brand tone.
3. Generate positioning.
4. Use the output to guide listing, ads, and images.

## Input Fields

Product, category, target customer, price, competitors, differentiators, use cases, brand tone, and sales channel.

## Output Fields

Customer profile, motivations, pain points, functional value, emotional value, differentiation, selling points, price position, messaging angles, and risks.

## Common Mistakes

- Targeting everyone
- Listing features without a buying reason
- Using unverifiable competitor assumptions

## Compliance Reminder

Do not turn guesses into facts. Mark assumptions clearly.
