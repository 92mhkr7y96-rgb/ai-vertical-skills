# Review Analysis Skill

Extracts ecommerce insights from provided review excerpts.

## Suitable Scenarios

- Own product review analysis
- Competitor review research
- Negative review retrospective
- FAQ planning
- Ad angle discovery

## Steps

1. Provide positive, negative, neutral, and competitor review excerpts.
2. Add product facts and platform.
3. Run the skill.
4. Review the theme table and recommended actions.

## Input Fields

Product category, platform, positive reviews, negative reviews, neutral reviews, competitor reviews, and product facts.

## Output Fields

Positive themes, negative themes, motivations, use scenarios, complaints, product suggestions, listing suggestions, ad angles, FAQs, claims to avoid, and summary table.

## Common Mistakes

- Providing too few review excerpts
- Mixing unrelated product categories
- Treating one review as a market-wide fact

## Compliance Reminder

Never fabricate reviews, ratings, or user quotes. Do not encourage review manipulation.
