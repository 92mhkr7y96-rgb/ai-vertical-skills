# Product Listing Optimizer

Optimizes ecommerce listing copy from provided product facts.

## Suitable Scenarios

- New product listing draft
- Old listing rewrite
- Amazon bullet improvement
- Shopify product page copy
- Etsy digital product listing

## Steps

1. Provide product facts and platform.
2. Add target customer, use case, tone, and constraints.
3. Run the skill.
4. Review claims before publishing.

## Input Fields

Product name, category, platform, customer, use case, features, specs, price range, tone, competitor notes, language, and compliance constraints.

## Output Fields

Positioning summary, title, title alternatives, benefits, feature-to-benefit map, bullets, description, keywords, FAQs, risky claims, and A/B variants.

## Common Mistakes

- Asking for claims not supported by product facts
- Mixing competitor claims into your own listing
- Using absolute wording such as "best", "guaranteed", or "never"

## Compliance Reminder

Verify all claims, certifications, trademarks, and platform rules before publishing.
