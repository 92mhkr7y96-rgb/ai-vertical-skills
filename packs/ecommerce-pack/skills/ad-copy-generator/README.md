# Ad Copy Generator

Creates ecommerce ad copy variants for platform-aware creative testing.

## Suitable Scenarios

- Meta ad testing
- TikTok hook testing
- Google ad headline ideation
- Amazon sponsored brand copy
- Promotion angle planning

## Steps

1. Provide product, platform, customer, pain point, benefit, offer, and restrictions.
2. Generate variants.
3. Select test groups.
4. Review compliance before launch.

## Input Fields

Product, platform, target customer, pain point, benefit, offer, tone, restrictions, landing page, and language.

## Output Fields

Hooks, short headlines, long headlines, primary text, CTAs, pain-point angle, scenario angle, comparison angle, promotion angle, A/B test plan, and risky wording.

## Common Mistakes

- Using fake discounts
- Claiming results that cannot be proven
- Ignoring platform style differences

## Compliance Reminder

Verify offers, discounts, claims, and targeting rules before running ads.
