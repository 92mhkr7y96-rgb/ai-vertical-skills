# Ecommerce Skill Pack 中文使用说明

本文档说明 Ecommerce Skill Pack 中 6 个核心 skill 的用途、推荐使用顺序、输入准备、输出内容、人工审核重点和合规边界。

## 1. Pack 定位

Ecommerce Skill Pack 是一套面向电商卖家和运营团队的开源 AI 工作流 Skill Pack。它用于把产品事实、市场记录、竞品观察和用户评价片段整理成结构化电商运营产物：

- 产品定位
- 竞品分析
- 用户评价分析
- 商品 Listing 优化
- 广告文案生成
- 商品图片 Brief 生成

适用场景包括 Amazon、Shopify、Etsy、TikTok Shop、独立站和社交电商运营。

## 2. 推荐使用流程

新品上架或老品优化时，建议按以下顺序使用：

1. `product-positioning-analyzer`
2. `competitor-product-analysis`
3. `review-analysis-skill`
4. `product-listing-optimizer`
5. `ad-copy-generator`
6. `product-image-brief-generator`

这个顺序可以让输出更稳：先明确产品定位和目标用户，再分析竞品机会，然后从评价中提炼用户语言，接着优化 Listing，再生成广告测试文案，最后把同一套卖点转成图片 Brief。

## 3. 6 个 Skill 详细说明

### 3.1 product-positioning-analyzer

用途：帮助卖家明确目标用户、购买动机、核心痛点、功能价值、情绪价值、差异化角度和主卖点。

适合场景：

- 新品定位
- DTC 品牌产品策略
- Listing、广告、图片素材前的卖点梳理
- 选品和产品差异化分析

需要准备的输入：

- 产品和类目
- 目标用户
- 价格或价格区间
- 竞品
- 差异化特征
- 使用场景
- 品牌语气
- 销售渠道

主要输出：

- 目标用户画像
- 购买动机
- 核心痛点
- 功能价值和情绪价值
- 差异化角度
- 1 个主卖点
- 3-5 个辅助卖点
- 价格定位
- 推荐表达角度
- 风险和弱点

人工审核重点：

- 目标用户是否足够具体
- 主卖点是否清楚
- 是否把推测写成事实
- 是否编造评价、认证、销量或性能数据

### 3.2 competitor-product-analysis

用途：分析竞品页面、卖点、价格策略、评价片段、页面强弱点，并提炼我方可突破机会。

适合场景：

- 竞品页面分析
- Listing 改版
- 差异化定位
- 广告角度规划
- 商品图优化

需要准备的输入：

- 竞品名称、标题、价格和平台
- 竞品 Bullet Points 和描述
- 竞品评价片段
- 竞品图片观察
- 我方产品记录

主要输出：

- 竞品定位
- 目标用户推测
- 主要卖点
- 价格策略
- 页面优势和弱点
- 基于评价的机会
- 差异化机会
- 我方 Listing 优化建议
- 广告角度机会
- 商品图片机会

人工审核重点：

- 结论是否来自用户提供的竞品信息
- 推测是否被明确标注
- 是否复制了竞品原文
- 建议是否可执行

### 3.3 review-analysis-skill

用途：从用户提供的评价片段中提炼满意点、差评痛点、购买动机、使用场景、产品改进建议、Listing 优化建议、广告角度和 FAQ 机会。

适合场景：

- 自有产品评价复盘
- 竞品评价研究
- 差评问题整理
- FAQ 规划
- 广告素材角度提炼

需要准备的输入：

- 产品类目
- 平台
- 好评
- 差评
- 中评
- 竞品评价
- 产品事实

主要输出：

- 高频好评主题
- 高频差评主题
- 常见购买动机
- 常见使用场景
- 重复投诉点
- 产品改进建议
- Listing 优化建议
- 广告文案角度
- FAQ 机会
- 应避免的声明
- 评价摘要表

人工审核重点：

- 是否只分析用户提供的评价
- 是否编造评分、原话或用户情绪
- 是否区分高频和低频问题
- 是否存在诱导刷评或压制差评的建议

### 3.4 product-listing-optimizer

用途：把产品事实转化为平台适配的标题、卖点、描述、关键词、FAQ 和 A/B 测试版本。

适合场景：

- 新商品 Listing 创建
- 老 Listing 改版
- Amazon 五点描述优化
- Shopify 商品页文案
- Etsy 商品标题和描述

需要准备的输入：

- 产品名称和类目
- 目标平台
- 目标用户
- 主要使用场景
- 核心功能
- 材料或规格
- 价格区间
- 品牌语气
- 竞品记录
- 输出语言
- 合规限制

主要输出：

- 产品定位摘要
- 优化标题
- 短标题备选
- 核心利益点
- 功能到利益转换表
- Bullet Points
- 商品描述
- 关键词建议
- FAQ 机会
- 需要避免的风险声明
- A/B 测试版本

人工审核重点：

- 是否把功能转化成用户利益
- 是否适配目标平台
- 所有声明是否有产品事实支撑
- 是否出现虚假评价、虚假认证或绝对化保证

### 3.5 ad-copy-generator

用途：生成适配不同广告平台的 Hook、标题、正文、CTA、角度版本和 A/B 测试计划。

适合场景：

- Meta 广告测试
- TikTok Hook 测试
- Google 广告标题探索
- Amazon 广告文案
- 促销角度测试

需要准备的输入：

- 产品
- 平台
- 目标用户
- 核心痛点
- 主要利益点
- Offer
- 语气
- 限制条件
- 落地页
- 输出语言

主要输出：

- Hook 选项
- 短标题
- 长标题
- 广告正文版本
- CTA 选项
- 痛点角度
- 场景角度
- 对比角度
- 促销角度
- A/B 测试计划
- 需要避免的风险措辞

人工审核重点：

- 文案风格是否匹配平台
- 折扣和 offer 是否真实
- 是否存在虚假稀缺、虚假紧迫感
- 是否包含无法证明的功效、健康、法律、金融、认证或性能承诺

### 3.6 product-image-brief-generator

用途：为商品主图、场景图、功能图、尺寸图、对比图、问题解决图和 A+ 内容图生成可执行的设计或拍摄 Brief。

适合场景：

- 商品图片组规划
- 给设计师或摄影师派单
- Amazon、Shopify、Etsy 图片改版
- 把 Listing 卖点转成视觉素材

需要准备的输入：

- 产品
- 平台
- 目标用户
- 核心功能
- 主要利益点
- 材料或规格
- 品牌风格
- 图片数量
- 竞品图片观察
- 合规限制

主要输出：

- 主图 Brief
- 场景图 Brief
- 功能图 Brief
- 尺寸/规格图 Brief
- 对比图 Brief
- 问题解决图 Brief
- A+ 内容图 Brief
- 图片文案建议
- Shot list
- 道具/背景建议
- 需要避免的内容

人工审核重点：

- 每张图的目标是否清楚
- 主图是否符合平台规则
- 图片文案是否短且合规
- 是否出现虚假徽章、虚假认证、评价数量、商标或未经证明的参数

## 4. 如何准备高质量输入

输入越真实，输出越安全、越可用。建议在使用前准备：

- 已确认的产品事实
- 准确规格和材料
- 平台限制
- 真实用户评价片段
- 真实竞品页面观察
- 品牌语气样例
- 必须避免的声明

不要要求 skill 自动补全缺失的认证、销量、评分、评价、测试数据或平台审核结果。

## 5. 输出发布前审核流程

任何输出发布前都需要人工审核：

1. 对照产品资料检查每一个声明。
2. 删除无法证明的认证、奖项、排名和性能承诺。
3. 核实平台规则和广告政策。
4. 检查商标和品牌授权。
5. 确认 offer、折扣、价格和物流信息真实。
6. 对保健品、护肤品、医疗器械、儿童用品、宠物食品、减肥产品等敏感类目进行更严格审核。

## 6. 安全合规规则

不要使用本 Pack：

- 生成虚假评价
- 生成虚假认证
- 编造销量
- 写医疗治疗承诺
- 写绝对化保证
- 伪造品牌授权
- 诱导刷评
- 生成仿牌、假货、高仿或侵权内容
- 承诺平台审核通过、广告审核通过、排名提升、转化提升或销售增长

## 7. 建议的案例文件结构
